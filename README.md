# IPO Offseason Insight Lab
공모주 서비스 비수기 사용자 행동을 데이터 기반으로 분석하고,
참여형 콘텐츠 전략 인사이트를 도출하기 위한 분석 프로젝트입니다.

본 레포는 내부 사용자 데이터 분석과
커뮤니티 리뷰 기반 텍스트 분석을 함께 수행하여
비수기 사용자 흐름을 다각도로 탐색하는 것을 목표로 합니다.


## 📂 Repository Structure

### 1️⃣ 공모주앱리뷰 (NLP Analysis)
사용자 리뷰 데이터를 기반으로 공모주 서비스 인식 구조를 분석합니다.

- 공모주 리뷰 데이터 수집 및 추출
- 텍스트 전처리 및 Word2Vec / 토픽 탐색


### 2️⃣ 비수기데이터분석 (Behavior Analysis)
비수기 사용자 행동 데이터를 구조화하고 참여 패턴을 분석합니다.

- 비수기 DAU 흐름 분석
- 체류시간 및 Tap Interaction Map 해석
- 커뮤니티 활동 데이터 분석
- HeatScore 기반 시즌 구분 및 지표 설계
- 

## 🧠 Key Focus

- 데이터 구조화 중심 분석 설계
- 지표 기반 시즌성 해석 (HeatScore)
- 데이터 QA 및 정합성 검증 관점 유지

본 프로젝트는 단순 분석 결과보다
데이터 설계 과정과 인사이트 도출 흐름을 기록하는 데 목적이 있습니다.
